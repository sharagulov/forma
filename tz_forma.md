# ТЗ: Сайт-пример для портфолио — «FORMA Architects»

> Лендинг-мультистраничник для вымышленной архитектурной студии.
> Цель: лучший проект в портфолио фронтенд-разработчика, который убедит клиентов заказать сайт.

---

## 1. Почему именно архитектурная студия

| Критерий | Почему подходит |
|---|---|
| **Визуальный WOW-эффект** | Архитектура = большие фото, минимализм, чистая типографика. Это тот тип сайтов, куда заходишь и думаешь «хочу такой же». Лучшие сайты мира (Awwwards, FWA) на 30%+ — архитектура и дизайн-студии. |
| **AI-генерируемый контент** | Тексты про «философию студии», описания проектов, биографии архитекторов — всё это ИИ пишет идеально. Никакой реальной базы клиентов или данных не нужно. |
| **Минимум ручной работы** | Нужны только фото (берём с Unsplash/Pexels — свободная лицензия) и деплой. Всё остальное генерируется агентами. |
| **Демонстрация навыков** | Плавные scroll-анимации, параллакс, кастомный курсор, адаптивная сетка, lazy-loading изображений — всё, что хочет видеть клиент. |
| **Универсальная привлекательность** | Красивый сайт архитектуры впечатляет ВСЕ ниши. Владелец ресторана, стоматолог, IT-стартап — все скажут «сделайте мне как у этих». |

## 5. Интерактивность и анимации

### 5.1. Обязательные эффекты

| Элемент | Анимация | Библиотека |
|---|---|---|
| **Scroll-reveal секций** | Fade-in + slide-up при скролле (IntersectionObserver) | CSS / GSAP ScrollTrigger |
| **Hero-заголовок** | Посимвольная или пословная анимация появления | GSAP SplitText или CSS @keyframes |
| **Hover на карточках проектов** | Scale фото (1.0 → 1.05), появление overlay с текстом | CSS transform + transition |
| **Навигация** | Скрытие при скролле вниз, появление при скролле вверх | JS + CSS transform |
| **Плавный скролл** | Smooth scroll с `scroll-behavior: smooth` + Lenis | Lenis (smooth scroll library) |
| **Кастомный курсор** | Кружок, следующий за мышью, меняющийся при hover | JS + CSS |
| **Page transitions** | Fade / slide между страницами | View Transitions API или GSAP |
| **Параллакс на hero** | Фоновое фото движется медленнее контента | CSS `transform: translate3d` |
| **Горизонтальный скролл** | Лента фото горизонтально при вертикальном скролле | GSAP ScrollTrigger horizontal pin |
| **Loader** | Минимальный прелоадер при первом заходе (линия + число %) | CSS + JS |

### 5.2. Правила анимаций

- **Длительность**: 0.4-0.8s для появлений, 0.2-0.3s для hover
- **Easing**: `cubic-bezier(0.22, 1, 0.36, 1)` — основной; `ease-out` — для hover
- **Reduce-motion**: Полностью отключать анимации при `prefers-reduced-motion: reduce`
- **Никогда**: не блокировать контент анимацией дольше 1.5 сек, не анимировать `width/height/top/left` (только `transform` и `opacity`)

## 6. Технический стек

### 6.1. Основные технологии

| Слой | Технология | Почему |
|---|---|---|
| **Фреймворк** | Next.js 15 (App Router, RSC) | SSR/SSG, быстрая загрузка, оптимизация изображений, лучшие Core Web Vitals |
| **Язык** | TypeScript 5.x | Строгая типизация, меньше багов, стандарт 2026 |
| **Стили** | Tailwind CSS 4.x + CSS custom properties | Быстрая разработка, минимальный CSS bundle |
| **Анимации** | GSAP 3.x (ScrollTrigger, SplitText) + Lenis | Лучшая производительность scroll-анимаций |
| **Изображения** | Next/Image + формат WebP/AVIF | Автоматическая оптимизация, lazy loading, responsive |
| **Шрифты** | next/font (Google Fonts) + Fontshare CDN | Нулевой Layout Shift |
| **Деплой** | Vercel (или Cloudflare Pages) | Edge SSR, глобальный CDN, бесплатный план |

### 6.2. Альтернатива (если без Next.js — чистый HTML/CSS/JS)

> Подходит, если цель — показать чистое владение фронтом без фреймворков.

| Слой | Технология |
|---|---|
| Сборка | Vite 6.x |
| Фреймворк | Нет (Vanilla JS / ES Modules) или Astro 5.x |
| Стили | Vanilla CSS + CSS Custom Properties (без препроцессоров) |
| Анимации | GSAP + Lenis |
| Деплой | Cloudflare Pages / Netlify / Vercel Static |

### 6.3. Производительность (целевые метрики)

| Метрика | Цель | Как достигается |
|---|---|---|
| **LCP** (Largest Contentful Paint) | < 1.5s | SSR/SSG, оптимизированные изображения, preload hero |
| **INP** (Interaction to Next Paint) | < 100ms | Минимальный JS в main thread, debounce scroll handlers |
| **CLS** (Cumulative Layout Shift) | < 0.05 | Явные размеры изображений, font-display: swap + size-adjust |
| **FCP** (First Contentful Paint) | < 0.8s | Edge CDN, минимальный render-blocking CSS |
| **JS Bundle** | < 150KB gzipped | Tree-shaking GSAP, динамический импорт |
| **Lighthouse Score** | 95+ (все категории) | Семантический HTML, a11y, оптимизированные ресурсы |

## Итог

Этот сайт при правильной реализации:
- Займёт **2-4 часа** чистой работы (из них ~1 час — подбор фото)
- Будет выглядеть как **$5,000-$10,000 коммерческий проект**
- Покажет клиенту: performance, анимации, адаптив, дизайн-систему, чистый код
- Подойдёт для Upwork, Freelancer, Toptal, и любого портфолио

Ниша «архитектурная студия» — золотой стандарт портфолио веб-разработчика, потому что красота сайта говорит сама за себя, а контент генерируется ИИ на 100%.
